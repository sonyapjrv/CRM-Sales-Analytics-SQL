# Real Estate CRM Analytics & System Design

## 🏢 Business Context
During my tenure at **Whitewill** and **AIM** (luxury real estate agencies), I managed operations and sales analytics for teams of 20-30 brokers. This project demonstrates the analytical framework I developed to move from manual Google Sheets reporting to a structured data-driven approach.

## 🛠️ Problems Solved
- **Manual Overhead:** Transformed time-consuming manual tracking into logical data models.
- **Conversion Tracking:** Designed a "Lead-to-Deal" funnel to identify bottlenecks in the sales process.
- **Quality Control:** Created a scoring system for call audits to improve broker-client communication.

## 📊 Key Metrics Implemented
- **Conversion Rate (CR):** From incoming lead to property showing and final transaction.
- **Agent Performance Index:** Based on revenue, square footage sold, and CRM hygiene (data accuracy).
- **Project Popularity Score:** Analyzing which developers and residential complexes show the best ROI.

## 💾 SQL Implementation
The included `analytics_queries.sql` file contains the logic for:
1. **Sales Funnel Analysis:** Grouping performance by managers.
2. **Revenue Deep-Dive:** Aggregating sales by project and area.
3. **Quality Assurance (QA) Metrics:** Tracking broker call scores for the Quality Control Department.

## 🚀 Impact
The implementation of these tracking systems allowed the management to:
- Adjust the motivation model based on transparent KPI data.
- Identify the most profitable real estate projects in the Moscow luxury market.
- Standardize CRM data entry through rigorous checklists and automated audits.
